    jrnl v0.1a

    This is an early alpha version of an open source journal application that runs entirely within the web browser. Your data is stored locally in your browser's database, and backup/restore features will soon be available to archive your data and back it up to other locations.

    There is no backend or database server for jrnl, so the developers of jrnl do not have any way to store or use your information. Your data stays with you at all times.

    For more information, visit the repository on github
